CSS:Cascading Style Sheets

Servers: these are dedidcated machines that serve out information to
client computers whenever its needed, they arealways online 24/7.

Tags : these are various elements in a markup lanuguage uses to tell the
how the text should be rendered. types of tags are:
-container tags : these have opening and closing tags
-self closing or empty tags:these types of tags dont need to have an opening and/or closing tag

Examples of tags

comment line: a container tag used for making documentations or code explanation
paragragh: a container tag used for making paragraghs
break line:this is a self closing tag used for moving to the next line
Strong : this is a container tag used for making text bold

Attributes or Properties: these are used to chsnge the default settings of tags and can be only used on opening tags

for example this line used the color property/attribute of the font tag to make it green

Javascripts - this give functionality to the code
